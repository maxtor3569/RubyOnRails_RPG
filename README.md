This application requires:
-------------

- Ruby 2.0.0
- Rails 4.2.0

Getting Started
---------------

  - Clone this repository

  ``` git clone https://github.com/chalas_r/RubyOnRails_RPG.git ```

  - Install gems

  ``` bundle install ```

  - Create database

  ``` rake db:create ```

  - Create Migrations

  ``` rake db:migrate ```

  - Start server

  ``` rails server ```

Enjoy !
